Chapter 6. System Requirements

  • Standard Intel architecture PC running Linux (tested on recent Red Hat, SuSE and Debian distributions), Microsoft Windows NT (no earlier than 4.0 + SP6a), Windows 2000 and Windows XP. Linux distributions from other vendors may also work, but are currently untested.

  • Enough disk space for the installed distribution. The eCos installation process will detail the various components of eCos and the compiler toolkit that can be installed, and their disk space requirements.

  • 64MB of RAM and a 350MHz or faster Pentium processor.

If you are downloading the eCos release distribution from sources.redhat.com/ecos, you will also need space to store that image and to compile the toolchain and eCos from source.
